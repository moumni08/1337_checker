**The new 1337 Checker for pool and check-in places**

## Install Pipenv:

```python
pip install pipenv
```
Python version: 3.9: https://www.python.org/downloads/release/python-392/

## Install:

```
git clone https://github.com/RyouYoo/1337_checker.git
cd 1337_checker/src
pipenv shell
pipenv install --dev
```

## Usage:

```python
python app.py "[EMAIL]" "[PASSWORD]"
```
